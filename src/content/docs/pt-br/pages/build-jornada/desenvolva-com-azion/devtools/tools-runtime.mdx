O **Azion Edge Runtime** é um conjunto de ferramentas para criar aplicações rápidas e responsivas que rodam próximas aos usuários.

Ele utiliza JavaScript e permite a utilização de várias APIs para tarefas como rede e manipulação de dados.

O Runtime suporta frameworks como o Next.js e incentiva as melhores práticas de codificação com o `Strict mode`.

## Overview

This document provides a JSON template to create a Grafana dashboard that displays data from the Real-Time Metrics GraphQL API, such as:

- Total Requests
- Cache Requests
- Status Codes
- Top 10 Request URI
- Top 10 Hosts

## Using the JSON Template

Follow the steps described in [How to query GraphQL data on Grafana Cloud](/en/documentation/products/guides/graphql-grafana-cloud/) and then use the following JSON with the `https://api.azionapi.net/metrics/graphql` URL to create a dashboard on Grafana.

## GraphQL Queries

To customize the data displayed on your Grafana dashboard, you can use GraphQL queries. Below is an example of a GraphQL query used in the JSON template:

```graphql
query HttpCalculatedTotalRequestsPerSecond {
 httpMetrics(
 limit: 1000
 filter: {
	tsRange: { begin: "${__from:date:iso}", end: "${__to:date:iso}" }
	 
		},
 groupBy: [ts]
 orderBy: [ts_DESC]
 ) 
 { 
 ts
 missedRequests
 requestsOffloaded
 requestsPerSecondOffloaded
 edgeRequestsTotalPerSecond
 missedRequestsPerSecond
 savedRequestsPerSecond
 httpRequestsTotal
 httpsRequestsTotal
 edgeRequestsTotal
 }
}
```

For more details on how to construct and modify GraphQL queries, refer to our [GraphQL Query Guide](/en/documentation/products/guides/graphql-query-guide/).

## Real-Time Metrics

The Real-Time Metrics API provides the following key metrics:

- **Total Requests:** The total number of requests received.
- **Cache Requests:** The number of requests served from cache.
- **Status Codes:** Breakdown of requests by status code.
- **Top 10 Request URI:** The most requested URIs.
- **Top 10 Hosts:** The most requested hosts.
